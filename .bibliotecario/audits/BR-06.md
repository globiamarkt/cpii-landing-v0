#### Resumen ejecutivo
`referrals.html` es una página de visualización de red y demostración de Revenue Share. Utiliza un árbol SVG estático (v1.1.0) para proyectar la estructura de 3 niveles de compensación del Club. Es una pieza clave de preventa para perfiles de Promotor/Embajador.

#### Contexto de repo
Estado: unificado — sincronizado con el chasis compartido.
Dependencias principales:
- `design-system.css` para estilos de nodos y scrollbars.
- `i18n.js` para la traducción de términos de red (Volumen, Comisión, Niveles).

#### Decisiones clave
- **Árbol SVG Manual:** Uso de coordenadas relativas (`%`) en el SVG para mantener la alineación de los conectores con los nodos (DIVs de posición absoluta), equilibrando flexibilidad visual y control de layout.
- **Dashboard de Realidad Aumentada:** Presentación de datos demo ("Demo User") que actúan como prototipo funcional de lo que el inversor verá tras la aprobación.

#### Potenciales conflictos y refactors sugeridos (para Agentes de dirección)
- **SVG vs DOM:** La sincronización de `line` SVG con `div` de nodos es frágil en redimensionamientos extremos. Se recomienda evaluar un motor de diagramas ligero o centralizar las coordenadas en un objeto JS si el árbol crece.
